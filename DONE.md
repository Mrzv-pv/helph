# HELPH — Что сделано

## Обзор
HELPH — маркетплейс профессиональных услуг. Платформа для поиска проверенных специалистов (юристы, финансисты, дизайнеры, маркетологи, IT и др.).

## Технологии
- **Next.js 16** (App Router) + TypeScript
- **Tailwind CSS v4** — стилизация
- **Lucide React** — иконки
- **CSS Variables** — дизайн-токены (цвета, шрифты, тени, радиусы)
- **Шрифты**: Inter (основной), Plus Jakarta Sans (заголовки)

## Дизайн-система
- Палитра: Jaguar Deep Green (#0D2818) + Gold (#B8962E) + Warm White (#F5F0E8)
- UI-компоненты: Button (5 вариантов), Input, Card, Badge, Avatar, Rating, LanguageSwitcher
- Логотип: буква H с молотком в золотом квадрате

## Страницы

### Маркетинг
- **/** — Лендинг: герой с поиском, "Как работает", категории, топ-специалисты, отзывы, CTA

### Каталог
- **/catalog** — Каталог специалистов с фильтрами и сортировкой
- **/catalog/[id]** — Профиль специалиста (услуги, отзывы, рейтинг)

### Авторизация
- **/auth/login** — Вход
- **/auth/register** — Регистрация

### Онбординг (5 шагов)
- **/auth/onboarding/role** — Выбор роли (специалист/клиент)
- **/auth/onboarding/profile** — Заполнение профиля
- **/auth/onboarding/services** — Создание услуги
- **/auth/onboarding/verify** — Верификация (email + документы)
- **/auth/onboarding/welcome** — Приветствие

### Платформа
- **/dashboard** — Панель управления (метрики, заказы, сообщения)

## Мультиязычность (i18n)
- **Словенский** (по умолчанию), Английский, Русский, Немецкий
- Клиентский i18n через React Context (`useI18n` хук)
- Переключатель языков в шапке с флагами
- 150+ ключей перевода
- Локализованные мок-данные (специалисты, услуги, отзывы)
- Валюта: EUR (SL/EN/DE), RUB (RU)
